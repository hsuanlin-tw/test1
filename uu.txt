# nano /etc/apt/apt.conf.d/10periodic

1.4 Setup network interface
a. Use the netplan to configure the network. The netplan is the default network tool on Ubuntu 22.04.2.

1.4 設定網路介面
a. 使用 netplan 設定網路。 netplan 是 Ubuntu 22.04.2 上的預設網路工具。

# sudo apt-get update 
# sudo apt install netplan.io netplan-

Note: If you have setup any record in the /etc/network/interfaces, please remove it before process this step.
注意：如果您在 /etc/network/interfaces 中設定了任何記錄，請在執行此步驟之前將其刪除。

b. Create a network configure file (please use Linux format)
b. 建立網路設定檔（請使用 Linux 格式）

# vim /etc/netplan/01-netcfg.yaml

network: version: 2 renderer: networkd ethernets: enp3s0: dhcp4: no addresses: - 192.168.100.100/24 routes: - to: default via: 192.168.100.1 nameservers: addresses: [8.8.4.4, 8.8.8.8]

# chmod 600 /etc/netplan/*.yaml

Note: Please make sure the interface name in the yaml file should be same as the above network configuration. For example here, the network interface name is “enp3s0”.
注意：請確保 YAML 檔案中的介面名稱與上述網路設定中的介面名稱一致。例如，此處的網路介面名稱為“enp3s0”。

c. Apply network configure
c. 應用網路配置

# netplan apply
